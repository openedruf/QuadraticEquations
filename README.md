# QuadraticEquations
Program solves roots for a quadratic equation.

# Installing
Before launching you need to install libraries.  <br/>
Go to your Terminal (MacOS) or cmd (Windows) and write:  <br/>

* pip install pyqt5
* pip install pyqt5-tools
